AlphaFold has changed the game for protein structure prediction and allows anyone with sufficient bioinformatics skills to predict the structure of virtualy any protein.

We ran AlphaFold via GoogleColab at: https://github.com/sokrypton/ColabFold

In particular, we used their AlphaFold2_mmseqs2 version that uses mmseqs2 rather than HMMMer for sequence search.

The main outputs include a set of PDB structure files along with matching JSON format files that tell us how good the resulting models might be. Letâ€™s start by loading these structures up in Mol*
